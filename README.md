
# React + TypeScript Example

## Install TypeScript and http-server

```
$ npm install -g typescript
$ npm install -g http-server
```

## Compile TypeScript source code

In project root,

```
$ tsc
```

## Startup http-server

In project root,

```
$ http-server web
```

Open [http://localhost:8080/](http://localhost:8080/) by web browser
